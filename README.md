# Python Bug Fix Examples

This repository contains real-world Python bug fixes with clear
**before and after** examples.

The goal of this repo is to demonstrate:
- Debugging skills
- Logical problem solving
- Clean fixes to common Python mistakes

Each example includes:
- A buggy version
- A fixed version
- A short explanation of the issue
